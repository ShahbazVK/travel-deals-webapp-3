# CS 6314 - Assignment #3
## TravelDeals - Premium Travel Booking Website

## Quick Start

1. **Start PHP Server**:
   ```bash
   php -S localhost:8000
   ```

2. **Open in Browser**:
   Navigate to: `http://localhost:8000/index.html`

## Project Structure

```
Assignment3/
├── (root) - HTML pages
├── css/ - Stylesheets
├── js/ - JavaScript files
├── php/ - PHP backend scripts
├── data/ - JSON/XML data files
└── docs/ - Documentation
```

- **7 HTML Pages**: index, stays, flights, cars, cruises, contact, cart (root)
- **CSS**: `css/mystyle.css` (external stylesheet)
- **JavaScript**: `js/app.js`, `js/dark-mode-init.js`
- **Data Files**: `data/flights.json` (93 flights), `data/hotels.xml` (22 hotels), `data/cars.xml` (22 cars)
- **PHP Backend**: 7 PHP scripts in `php/` folder for file updates and booking storage
